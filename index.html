Tutorial

This is a Simulator-operate tutorial, not a tutorial for the Injury CPU Architecture / Assembly

All ASCII keys are captured, except for the control keys, those had issues implementing, so you can't really input 0x01 - 0x1F, except for 0x0A (Enter) and 0x1B (Escape, might not work in fullscreen).

To open the debug info menu, press CONTROL+I

To capitalize letters, press SHIFT for capitalizing one character, or press CAPS LOCK, to switch between default lowercase and uppercase, also you can press the capslock button ("abc" displayed if capslock is off, "ABC" if on)
